    CSS egenskaber

    Nye og gamle egenskaber

    CSS er som bekendt under W3C og de sørger for løbende af opdatere og frigive nye moduler. Det kan dog nogen gange være lidt en udfordring at finde ud af hvilke egenskaber som med sikkerhed kan bruges. Det skyldes især også, at de nye browsere kan have implementeret en egenskab lang tid inden den er i REC fra W3C.

    Alle REC moduler kan uden problemer benyttes og andre moduler kan man få et overblik over på https://www.caniuse.com (se elsempelvis sticky. At canuise.com siger god for en egenskab betyder dog ikke nødvendigvis, at egenskaberne er implementeret og man bør derfor altid foretage test - eventuelt ved hjælp af sites som https://www.browserstack.com/.

    CSS Vendor Prefixes

    Ved brug af de helt nye egenskaber skal der eventuelt benyttes et prefix for at fortælle browseren at man ønsker en ny feature. Her er eksempelvis prefix brugt i forbindelse med animation.

    p {
    -webkit-transition: all 4s ease;
    -moz-transition: all 4s ease;
    -ms-transition: all 4s ease;
    -o-transition: all 4s ease;
    transition: all 4s ease;
}

    Browserne har forskellige koder:

    • Android: -webkit-
    • Chrome: -webkit-
    • Firefox: -moz-
    • Internet Explorer: -ms-
    • iOS: -webkit-
    • Opera: -o-
    • Safari: -webkit-

    Det er ikke så tit du ser brug af prefix efterhånden, men et site som https://autoprefixer.github.io/ giver en nem mulighed for at sikre at man benytter de prefix man skal.

    Tekster

    Der er et hav af egenskaber relateret til tekster.

    Fonte

    Alle de interne fonte i browseren kan benyttes direkte med ‘font-family’. Hvis en font ikke findes faldes der tilbage på andre mere generelle fonte.

    p {
    font-family: "Segoe UI", Tahoma, Geneva, Verdana, sans-serif;
}

    Men det er også muligt at benytte eksterne fonte

    • WOFF-format
    • True Type-format

    De kan eksempelvis hentes hos Google fonts ved hjælp af @font-face:

    @font-face {
    font-family: "Shadows Into Light";
    font-style: normal;
    font-weight: 400;
    src: local("Shadows Into Light"), local("ShadowsIntoLight"),
        url(https://fonts.gstatic.com/s/shadowsintolight/v7/UqyNK9UOIntux_czAvDQx_ZcHqZXBNQzdcD5.woff2)
            format("woff2");
    unicode-range: U+0000-00FF, U+0131, U+0152-0153, U+02BB-02BC, U+02C6,
        U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2122, U+2191, U+2193,
        U+2212, U+2215, U+FEFF, U+FFFD;
}

p {
    font-family: "Shadows Into Light", sans-serif;
}

    Størrelse

    Størrelse er ikke bare relateret til tekster men også til bokse mv. De kan angives på flere måder

    • px
      • pixel
    • %
      • procent
    • em
      • relativt til mor
        • 2em (2 x mor)
        • 0.5em (1/2 mor)
    • rem
      • relativ til rod
    • pt
      • points
    • vw
      • relativt til viewport

    I relation til tekster benyttes px og eventuelt efterfølgende em/rem.

    section {
    font-size: 12px;
}

section p {
    font-size: 0.8em;
}

    Style

    Der findes en masse opmærkninger relateret til styling af tekst

    • font-style
      • italic
    • font-weight
      • bold
    • font-variant
      • small-caps
    • text-transform
      • lowercase
      • uppercase
    • text-decoration
      • underline
    • text-shadow
      • h-shadow, v-shadow, blur, color
    • text-align
      • left, center, right
    • vertical-align
      • top
      • middle
      • bottom
    • letter-spacing
      • px
    • word-spacing
      • line-height
    .speciel {
    font-style: italic;
    font-weight: bold;
    text-transform: uppercase;
    text-decoration: underline;
    text-shadow: 4px 4px 4px red;
}

    Kolonner

    Opdeling af tekst i kolonner er blevet nemt i de nyere browsere med ‘columns’:

    div#col {
    width: 500px;
    columns: 4;
    column-gap: 20px;
}

    Opgaver

    Følgende opgaver er tilgængelige:

    • Tekster og fonte - I denne opgave skal du arbejde lidt med tekster, fonte med videre.

    Farver

    Forskellige elementer har egenskaber relateret til farver - eksempelvis

    • tekst
      • color
      • background-color
      • text-shadow
    • box
      • box-shadow
      • border-color

    og farve kan angives på flere måder - bla.

    • ved navn
      • blue, red, green mv
    • ved hex
      • #ff000080, #0000ff80
    • ved RGB rgb(255, 0, 0), rgb(0, 0, 255)
    • ved HSL
      • hsl(120, 100%, 50%), hsl(120, 60%, 70%) mv

    Her er et par eksempler:

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Demo</title>
    <style>
      h1:nth-child(1) {
        color: red;
      }
      h1:nth-child(2) {
        color: rgb(0, 0, 255);
      }
      h1:nth-child(3) {
        color: hsl(120, 100%, 50%);
      }
    </style>
  </head>
  <body>
    <h1>Test</h1>
    <h1>Test</h1>
    <h1>Test</h1>
  </body>
</html>

    Box model

    Alle blok elementer kan defindes som en boks:

    billede ref

    En boks har forskellige egenskaber der kan benyttes

    • margin
    • border
    • padding
    • height
    • width
      • max-width
    • border-radius
    • box-shadow

    Her er et eksempel:

    div#box {
    width: 100px;
    height: 100px;
    background-color: yellow;
    border: 1px solid black;
    padding: 5px;
    margin: 5px;
    border-radius: 10px;
    box-shadow: 10px 10px 10px black;
}

    Opgaver

    Følgende opgaver er tilgængelige:

    • Box-modellen - Simpel opgave i brug af box-modellen i CSS

    Synlighed

    Der kan tit være behov for at skjule/vise elementer

    • display
      • skjul/vis element (tilføjer/fjerner element i rendering)
    • visibility
      • skjul/vis element (viser/skjuler blot)
    div#display {
    /* display: none; */
    visibility: hidden;
}

    Positionering

    Blok-elementer kan placeres i forhold til hinanden på forskellig vis.

    position-egenskaben

    • static
    • relative
      • relativ til den oprindelige position
    • absolute
      • realtiv til mor
    • fixed
    • sticky

    Placering

    • top/bottom
    • left/right
    • z-index

    Her er et eksempel på en boks placeres relativt.

    div#pos {
    position: relative;
    left: 20px;
    top: 20px;
    z-index: -1;
}

    Float

    Float kan benyttes til at placere elementer i forhold til hinanden (venstre/højre). Prøv følgende html

    <!-- https://developer.mozilla.org/en-US/docs/Web/CSS/float -->
<section>
  <div class="left">1</div>
  <div class="right">2</div>
  <p>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi tristique sapien ac erat tincidunt, sit amet dignissim lectus vulputate. Donec id iaculis velit. Aliquam vel malesuada erat.
    Praesent non magna ac massa aliquet tincidunt vel in massa. Phasellus feugiat est vel leo finibus congue.
  </p>
</section>

    med følgende css

    section {
    border: 1px solid blue;
}

section div {
    margin: 5px;
    width: 50px;
    height: 50px;
}

section .left {
    float: left;
    background: pink;
}

section .right {
    float: right;
    background: cyan;
}

    Flexbox

    En flexbox er en nyere måde at placere elementer i forhold til hinanden, og giver en masse muligheder for

    • plads mellem elementer
    • sortering
    • flow
    • justering

    Prøv eksempelvis https://flexbox.help/ eller dette eksempel på en simpel flexbox:

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Flex</title>
    <style>
      .box div {
        width: 50px;
        height: 50px;
        margin: 10px;
        background-color: antiquewhite;
      }

      .box {
        width: 300px;
        height: 300px;
        display: flex;
        flex-direction: row;
        /* flex-direction: row-reverse; */
        /* flex-direction: column; */
        /* flex-direction: column-reverse;  */
        background-color: red;
      }
    </style>
  </head>
  <body>
    <div class="box">
      <div>1</div>
      <div>2</div>
      <div>3</div>
    </div>
  </body>
</html>

    Prøv at rette flex-direction.

    Et andet eksempel er det klassiske holy-grail-of-design - check https://codepen.io/takeradi/pen/MyYqKX eller

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Holy Grale</title>
    <style>
      body {
        display: flex;
        flex-direction: column;
        min-height: 100px;
        text-align: center;
      }

      body .header {
        width: 100%;
        height: 60px;
        background: pink;
      }
      body .holygrail-body {
        flex: 1 0 auto;
        display: flex;
      }
      body .holygrail-body .content {
        flex: 1 0 auto;
        background: lightgreen;
      }
      body .holygrail-body .nav {
        width: 100px;
        list-style: none;
        text-align: left;
        order: -1;
        background: yellow;
        margin: 0;
      }
      body .holygrail-body .aside {
        width: 100px;
        background: orange;
      }
      body .footer {
        width: 100%;
        height: 60px;
        background: cyan;
      }
    </style>
  </head>
  <body>
    <header class="header">Header</header>
    <div class="holygrail-body">
      <div class="content">Content</div>
      <ul class="nav">
        <li>Menu 1</li>
        <li>Menu 2</li>

      </ul>
      <div class="aside">Aside</div>
    </div>

</html>

    Responsive design

    Responsive design handler om at skabe et design som er baseret på en aktuel tilstand i browseren som eksempelvis størrelse af viewport, rendering til browser/print eller konkret opløsning.

    Media typer

    Oprindeligt startede det med media typer som bruges til at skelne mellem desktop/print. Se følgende eksempel:

    <div id="mtype">
    Lorem, ipsum dolor sit amet consectetur adipisicing elit.
</div>

    med følgende css:

    div#mtype {
    color: red;
}

@media print {
    div#mtype {
        color: black;
    }
}

    Media queries

    I den mere avancerede form kan media queries bruges til at skelne mellem forskellige størrelser og opløsninger. Prøv følgende:

    <div id="qtype">
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestias fuga ad maiores nesciunt
    voluptatibus facere reiciendis. Beatae repellendus, sint provident iusto distinctio cumque
    voluptatibus deleniti cupiditate eveniet nobis iure eos!
</div>

    med denne css:

    div#qtype {
    color: black;
}

@media only screen and (min-width: 400px) {
    div#qtype {
        color: red;
    }
}

@media only screen and (min-width: 800px) {
    div#qtype {
        color: blue;
    }
}

@media only screen and (min-width: 1000px) {
    div#qtype {
        color: green;
    }
}

    Se University of Vermont som har et flot responsive design.

    Opgaver

    Følgende opgaver er tilgængelige:

    • Mediatyper - I denne opgave skal du prøve at lege med den simple form for dynamisk css
    • Leg med media queries - I denne opgave skal du bruge media queries til ændre farve afhængig af browserens størrelse.

    Animationer

    Animationer er en del af moderne css og erstatter til en hvis grad brug af JavaScript.

    Transforms

    Brug af transforms ligger til grund for animationer og gør det muligt at skalere, rotere, flytte og vride elementer:

    • scale
    • rotate
    • translate
    • skew

    Prøv følgende:

    <div id="transf">
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestias fuga ad maiores nesciunt
    voluptatibus facere reiciendis. Beatae repellendus, sint provident iusto distinctio cumque
    voluptatibus deleniti cupiditate eveniet nobis iure eos!
</div>

    med denne css:

    div#transf {
    width: 400px;
    transform: rotate(-5deg);
}

    Transitions

    Transforms kan så efterfølgende benyttes i transitions:

    billed ref

    Prøv følgende:

    <div id="trans">
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestias fuga ad maiores nesciunt
    voluptatibus facere reiciendis. Beatae repellendus, sint provident iusto distinctio cumque
    voluptatibus deleniti cupiditate eveniet nobis iure eos!
</div>

    med dette css:

    div#trans {
    width: 400px;
    background-color: yellow;
    transition-duration: 2s;
}

div#trans:hover {
    background-color: red;
}

    KeyFrame Animation

    KeyFrame Animations er næste skridt og giver mulighed for at skabe en komplet animation

    <div id="kf">
    Lorem ipsum
</div>

    Prøv følgende css:

    div#kf {
    width: 100px;
    height: 100px;
    position: relative;
    animation: boxani 5s;
}

@keyframes boxani {
    0% {
        background: green;
        left: 0px;
        top: 0px;
    }
    25% {
        background: blue;
        left: 150px;
        top: 0px;
    }
    50% {
        background: red;
        left: 150px;
        top: 150px;
    }
    75% {
        background: yellow;
        left: 0px;
        top: 150px;
    }
    100% {
        background: green;
        left: 0px;
        top: 0px;
    }
}

    Beregninger

    Beregninger kan ske med calc. Her et responsive banner:

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Banner</title>
    <style>
      .banner {
        position: absolute;
        left: 90px;
        width: calc(100% - 180px);
        border: solid black 1px;
        background-color: thistle;
        text-align: center;
    </style>
  </head>
  <body>
    <div class="banner">Banner!</div>
  </body>
</html>

    Attribut funktioner

    attr kan eksempelvis benyttes til at benytte værdien fra en attribut:

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Attr demo</title>
    <style>
      a::after {
        content: "[" attr(href) "]";
      }
    </style>
  </head>
  <body>
    <a href="https://www.google.dk"></a>
  </body>
</html>
